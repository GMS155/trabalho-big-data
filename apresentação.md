Apresentação:

Sistema distribuído de análise de telemetria veicular para monitoramento de comportamento de condução e eficiência operacional.

“Utilizou-se um subconjunto dos dados reais como amostra de um cenário de telemetria veicular para realizar o trabalho de Big Data”

Característica do Big Data (Volume):

Um veículo pode mandar dados a cada segundo:

1 evento/segundo

100 mil veículos:

100 mil eventos/segundo

1 dia:

8,64 bilhões de eventos

Dessa forma evidencia-se um problema de Big Data.

Dataset utilizado:
https://www.kaggle.com/datasets/yashdev01/vehicle-energy-and-telemetry-dataset
